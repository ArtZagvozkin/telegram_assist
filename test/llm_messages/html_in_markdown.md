# Приветствие и Введение

Это пример текста, который **смешивает** обычный синтаксис **Markdown** с встроенными элементами HTML.

## Разделы с Форматированием

### Маркдаун

*   Списки (неупорядоченные)
*   **Жирный текст**
*   *Курсив*

### Встроенный HTML

Мы можем использовать теги для более точного контроля над стилем.

<p style="color: blue; font-size: 1.2em;">
    Этот абзац стилизован с помощью встроенного <code>&lt;p&gt;</code> тега и инлайнового CSS.
</p>

<div class="warning-box">
    <h4>Важное Замечание</h4>
    <p>Внутри этого <code>&lt;div&gt;</code> мы можем размещать как обычный текст, так и другой отформатированный контент.</p>
    <span style="font-weight: bold; color: red;">Внимание!</span>
</div>

## Таблица (Markdown с Элементом HTML)

| Заголовок 1 | Заголовок 2 |
| :--- | :--- |
| Данные А | <b style="color: green;">Успех</b> |
| Данные Б | Обычный текст |

---

## Заключение

Markdown отлично подходит для быстрой разметки, но когда требуется специфическое визуальное оформление, **встраивание HTML** становится необходимым.
