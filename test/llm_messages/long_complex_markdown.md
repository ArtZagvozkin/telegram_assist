## 5. Код и Математика

### 5.1. Блоки Кода

Ниже представлен пример кода на Python, демонстрирующий сложный цикл:

````python
def calculate_fibonacci(n):
    """Вычисляет n-е число Фибоначчи."""
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

if __name__ == "__main__":
    result = calculate_fibonacci(10)
    print(f"Fib(10) = {result}")
````

### 5.2. Формулы (LaTeX/KaTeX)

Мы проверяем поддержку встроенных и блочных математических выражений.

**Встроенная формула:** Квадратное уравнение решается с помощью формулы $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$.

**Блочная формула:**
$$\Gamma(z) = \int_0^\infty t^{z-1} e^{-t} dt$$

Или более сложная матрица:
$$
\begin{pmatrix}
\cos \theta & -\sin \theta \\
\sin \theta & \cos \theta
\end{pmatrix}
$$
